# 🔐 SentinelPass

**SentinelPass** is a password security analysis and management tool that helps users evaluate and improve the strength of their passwords. The application analyzes passwords using entropy calculations, AI-based scoring, and breach detection to provide real-time security insights.

It also includes a dashboard where users can generate, analyze, and manage passwords along with their security metrics.

---

## 🚀 Features

* 🔑 **Password Strength Analyzer**
  Analyze passwords using entropy calculations and AI-based strength prediction.

* 🧠 **Security Insights**
  Displays password strength levels such as *Very Weak, Weak, Good, Better, and Very Strong*.

* ⏳ **Crack Time Estimation**
  Estimates how long a password would take to crack using brute-force attacks.

* 🔍 **Breach Detection**
  Checks passwords against known compromised password databases.

* 🎲 **Secure Password Generator**
  Generates strong random passwords with high entropy.

* 📊 **Interactive Dashboard**
  Save, manage, and review password security metrics.

* 🔐 **Authentication System**

  * Email registration with OTP verification
  * Secure login with hashed passwords
  * Google OAuth login

* 📁 **Password History Management**

  * Save analyzed passwords
  * Edit or delete entries
  * Export password history as CSV

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/sentinelpass.git
cd sentinelpass
```

Create virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
